# Contributing

By submitting a pull request to this repository, you agree that your
contribution will be licensed under the Apache License, Version 2.0.

Before opening a pull request:

- follow the architectural constraints in `.github/copilot-instructions.md`
- keep changes aligned with the documents in `docs/`
- avoid unrelated refactors or broad formatting-only changes
- include tests or validation steps when the change affects behavior

Pull requests should be focused, reviewable, and consistent with the modular
monolith, DTO, determinism, and database adapter rules used by this framework.
